The drosophila central complex functional connectivity atlas
This website contains data associated with the paper "Building a functional connectome of the Drosophila central complex" (Franconville, Beron and Jayaraman). Raw data is available as an Open Science Foundation project here. There's also a Github repository for the analysis code. Code notebooks can be run online with Binder here.
